# Saudi-Innovation-Forecast-Platform
Project Description
This project is a web-based platform developed as part of the coursework for the Saudi Innovation Forecast Platform. The system allows users to register, submit innovative ideas, analyze them using AI-based evaluation, and interact with ideas through engagement features. Firebase is used as a cloud backend for authentication and data storage.
